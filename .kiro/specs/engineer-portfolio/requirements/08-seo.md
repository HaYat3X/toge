# 要件 8: SEO・パフォーマンス

**Objective:** サイト運営者として、検索エンジンにおける適切なインデックスとページパフォーマンスを確保したい。そうすることで、潜在クライアントがサイトを発見しやすくなる。

## Acceptance Criteria

1. The Portfolio Site shall 全ページに適切なtitleタグ・meta descriptionタグを設定する。
2. The Portfolio Site shall OGP（Open Graph Protocol）タグを全ページに設定し、SNSシェア時に適切なプレビューを表示できるようにする。
3. The Portfolio Site shall `sitemap.xml` を生成し、検索エンジンのクローリングを補助する。
4. The Portfolio Site shall Core Web Vitals（LCP・CLS・FID）において良好なスコアを達成する設計とする。
5. The Portfolio Site shall Next.js Image コンポーネントを使用して画像を最適化し、WebP形式での配信とLazy Loadingを実現する。

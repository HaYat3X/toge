# Product Overview

**toge** は Next.js ベースの Web アプリケーションです。現在はプロジェクト初期段階であり、コア機能の定義と実装が進行中です。

## Core Capabilities

- モダン Web UI の提供（App Router + React Server Components）
- shadcn/ui ベースのデザインシステムによる一貫したコンポーネント体験
- ダークモード対応のレスポンシブレイアウト

## Target Use Cases

プロジェクト初期のため、ユースケースは仕様フェーズで定義される予定です。  
`.kiro/specs/` の仕様書を参照してください。

## Value Proposition

- 型安全な TypeScript + Next.js App Router による堅牢な基盤
- shadcn/ui パターンによる拡張性の高い UI コンポーネント設計
